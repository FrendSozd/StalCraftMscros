# ИНСТРУКЦИЯ ПО УСТАНОВКЕ

## ⚠ Важно: Антивирус может ругаться

Макрос использует эмуляцию клавиатуры (библиотеки `pydirectinput` и `keyboard`), 
что антивирусы иногда принимают за поведение кейлоггера или чита. 
Это **ложное срабатывание** — макрос НЕ собирает ваши данные и НЕ отправляет их.

## Как добавить в исключения Windows Defender

1. Откройте **Пуск → Параметры** (значок шестерёнки)
2. Перейдите в **Конфиденциальность и защита → Защита от вирусов и угроз**
3. В разделе "Параметры защиты от вирусов и угроз" нажмите **Управление настройками**
4. Прокрутите вниз до раздела **Исключения** → нажмите **Добавление или удаление исключений**
5. Нажмите **Добавить исключение** → выберите **Файл**
6. Укажите путь к `StalCraftX_Macro.exe`
7. Готово! Теперь Defender не будет его трогать

## Как добавить в исключения Kaspersky

1. Откройте Kaspersky → **Настройки** (шестерёнка)
2. Раздел **Угрозы и исключения** → **Исключения**
3. Нажмите **Управление исключениями** → **Добавить**
4. Укажите путь к `StalCraftX_Macro.exe`
5. Сохраните

## Как добавить в исключения Avast / AVG

1. Откройте Avast/AVG → **Меню** → **Настройки**
2. Раздел **Защита** → **Основные компоненты защиты** → **Щит файловой системы**
3. Перейдите на вкладку **Исключения**
4. Нажмите **Добавить** → укажите путь к `StalCraftX_Macro.exe`
5. Сохраните

## Если SmartScreen блокирует запуск

При первом запуске Windows может показать окно "Система Windows защитила ваш ПК".

1. В этом окне нажмите **Подробнее**
2. Появится кнопка **Выполнить в любом случае** → нажмите её
3. Файл запустится

## Если Edge/Chrome блокирует скачивание

1. В адресной строке нажмите на значок блокировки (или "..." → Скачивания)
2. Выберите **Сохранить** или **Сохранить в любом случае** / **Keep**
3. Или скачайте через Telegram (отправьте файл себе в Saved Messages)

## Активация макроса

1. Запустите `StalCraftX_Macro.exe` от имени Администратора
2. Введите лицензионный ключ в формате `SCX-XXXX-XXXX-XXXX-XXXX`
3. Ключ привяжется к вашему ПК (HWID)
4. Готово! Можно пользоваться

## Возможные проблемы

### "НЕ ХВАТАЕТ БИБЛИОТЕК"
Такого быть не должно — все библиотеки уже встроены в .exe.
Если всё же появилась ошибка — скачайте файл заново.

### "Нет связи с сервером лицензий"
Проверьте интернет-соединение. Макрос перейдёт в офлайн-режим 
и продолжит работать (с ограничениями).

### "Лицензия привязана к другому ПК"
Один ключ = один компьютер. Если вы сменили ПК или 
переустановили Windows — обратитесь к автору для переноса лицензии.

### "Срок действия лицензии истёк"
Ваша лицензия закончилась. Обратитесь к автору для продления 
или получения нового ключа.
