# StalCraft X Macro

Защищённый макрос для StalCraft X с привязкой лицензии к оборудованию (HWID).

## Возможности

- 🔐 Привязка лицензии к железу (CPU + диск + MAC)
- ⏱ Временные лицензии (сутки / неделя / месяц / год / навсегда)
- 🔄 Онлайн-проверка каждые 5 минут
- 🛡 Защита от отладки и тамперинга
- 🔑 Управление лицензиями через веб-админку

## Скачать

Последнюю версию можно скачать в [Releases](../../releases).

## Установка

1. Скачайте `StalCraftX_Macro.exe` из [Releases](../../releases/latest)
2. Если антивирус ругается — это ложное срабатывание (см. [ИНСТРУКЦИЯ.md](ИНСТРУКЦИЯ.md))
3. Добавьте файл в исключения Windows Defender
4. Запустите от имени Администратора
5. Введите лицензионный ключ (формат: `SCX-XXXX-XXXX-XXXX-XXXX`)



## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| F6 | Пауза / Продолжить |
| F9 | Выход |
